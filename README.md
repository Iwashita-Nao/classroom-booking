# 🏫 教室予約管理システム

教室の利用予約を簡単に管理できる Web アプリです。

## 機能

- **教室の追加** — 教室名・予約日・開始時刻を入力して予約を登録
- **利用時間の選択** — 30分・1時間・1時間30分・2時間から選択
- **重複チェック** — 同じ教室名・同じ日付での二重予約を防止
- **予約一覧** — 日付・時刻順に並び替えて表示
- **絞り込み検索** — 教室名でリアルタイム絞り込み
- **削除確認** — 誤削除を防ぐ確認モーダル
- **データ永続化** — `localStorage` でブラウザを閉じても保持

## 技術スタック

| 分類 | 使用技術 |
|------|----------|
| 構造 | HTML5 (Semantic) |
| スタイル | Vanilla CSS (Glassmorphism, CSS Custom Properties) |
| ロジック | Vanilla JavaScript (ES2020+) |
| フォント | Google Fonts / Noto Sans JP |
| ストレージ | localStorage |

## 使い方

1. `index.html` をブラウザで開く（またはローカルサーバーで起動）
2. 教室名・予約日・開始時刻・利用時間を入力して「予約を追加」
3. 一覧で登録済みの予約を確認・削除
